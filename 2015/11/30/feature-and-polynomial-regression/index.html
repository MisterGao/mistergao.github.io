机器学习（06）特征和多项式回归

  • 多项式回归
  • 机器学习
  • 笔记

线性回归不能适用于所有数据，有时候我们需要曲线来适应我们的数据，比如二次方模型：

$$h_\theta (x) = \theta_0 + \theta_1x_1 + \theta_2x_2^2$$

或者需要一个三次方模型：
$$h_\theta (x) = \theta_0 + \theta_1x_1 + \theta_2x_2^2 + \theta_3x_3^3$$

我们可以使：
$x_2 = x_2^2$
$x_3 = x_3^3$
从而使模型转化为线性回归模型。
根据图形的特点我们还可以得到如下模型

$$h_\theta (x) = \theta_0 + \theta_1(size) + \theta_2(size)^2$$

或者：

$$h_\theta (x) = \theta_0 + \theta_1(size) + \theta_2 \sqrt{size}$$

采用多项式回归模型时，在运行梯度下降法前，特征缩放是非常有必要的。

代价函数的定义跟之前的相似，建模的预测值与实际值的平方差之和。
